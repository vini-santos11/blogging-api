# Blogging API

API do Grupo 3 da PÓS Graduação Full Stack Development - FIAP

## Requisitos

- Node.js
- Docker

# Inicialização

1. Instale as dependências:

```shell
    npm install
```

2. Inicialize o banco de dados:

```shell
    docker compose up -d
```

3. Rode as migrations:

```shell
    npm run migration:run
```

4. Inicialize o projeto:

```shell
    npm start
```
